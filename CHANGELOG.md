# Change Log

All notable changes to Git Author Explorer will be documented in this file.

## [0.1.0] - 2026-05-15

### Added

- Add Git author search by name or email.
- Add multi-author selection with a date range filter.
- Add changed-file and dependency-file tree views.
- Add file open and diff commands from the side bar.
- Add English and Korean localization bundles.
