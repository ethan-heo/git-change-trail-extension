# 요구사항 개발 계획서

## 목표

VSCode에서 열린 파일의 Git 커밋 이력을 사이드바에서 탐색하고, 특정 커밋의 변경 파일을 디렉토리 트리로 확인하며, 커밋 간 파일 내용을 비교할 수 있게 한다.

## 주요 사용자 시나리오

1. 사용자가 에디터에서 파일을 열거나 탭을 전환하면 사이드바 패널이 해당 파일의 커밋 이력으로 자동 갱신된다.
2. 커밋 이력은 날짜 기준 내림차순으로 번호가 붙어 나열된다.
3. 커밋 항목을 클릭하면 해당 커밋에서 변경된 파일들이 디렉토리 트리로 표시된다.
4. 트리의 파일을 클릭하면 해당 커밋 시점의 파일 내용이 읽기 전용으로 열린다.
5. 커밋 파일 트리의 파일 항목 버튼으로 해당 파일의 전체 변경 이력을 Commit History 패널에 표시할 수 있다.
6. 커밋 파일 트리의 파일 항목 버튼으로 이전/이후 커밋과 diff를 열 수 있다.
7. 날짜 필터를 적용해 특정 기간의 커밋만 볼 수 있다.
8. 저자 필터를 적용해 선택한 저자의 커밋만 볼 수 있다.
9. 패널을 특정 파일에 고정(핀)해 탭 전환에도 이력이 바뀌지 않도록 할 수 있다.

## MVP 기능

| ID | 기능 | 구현 기준 |
|----|------|-----------|
| F-01 | 에디터 파일 자동 추적 | `onDidChangeActiveTextEditor` 이벤트로 활성 파일이 바뀌면 커밋 이력 패널을 자동 갱신한다. |
| F-02 | 우클릭 컨텍스트 메뉴 | Explorer에서 파일 우클릭 시 "커밋 이력 보기" 메뉴를 제공한다. |
| F-03 | 커밋 이력 목록 표시 | 파일의 커밋 이력을 날짜 내림차순으로 표시하고 `#1`, `#2` 형태로 번호를 붙인다. |
| F-04 | 날짜 기준 필터 | 패널 타이틀 아이콘 클릭 시 from/to 날짜 입력, 최근 기간 프리셋, 필터 초기화를 제공한다. |
| F-05 | 패널 핀 고정 | 핀 버튼 클릭 시 탭 전환에도 패널이 현재 파일에 고정된다. |
| F-06 | 커밋 클릭 → 변경 파일 트리 | 커밋 항목 클릭 시 해당 커밋에서 변경된 파일을 디렉토리 트리로 표시한다. |
| F-07 | 파일 내용 열기 | 커밋 파일 트리의 파일 클릭 시 해당 커밋 시점의 파일 내용을 실제 파일과 구분되는 제목의 읽기 전용 문서로 연다. |
| F-08 | 파일 전체 변경 이력 보기 | 커밋 파일 트리의 파일 항목 버튼에서 해당 파일의 전체 커밋 이력을 Commit History 패널에 표시한다. |
| F-09 | 이전 커밋과 diff | 커밋 파일 트리의 파일 항목 버튼에서 이전 커밋과 diff를 열 수 있다. |
| F-10 | 이후 커밋과 diff | 커밋 파일 트리의 파일 항목 버튼에서 이후 커밋과 diff를 열 수 있다. |
| F-11 | diff 비교 상태 표시 | diff 진행 중 커밋 파일 패널 타이틀에 `#N ↔ #M` 형태로 비교 상태를 표시한다. |
| F-12 | 빈 결과 처리 | 커밋 이력이 없거나 변경 파일이 없는 경우 안내 메시지를 표시한다. |
| F-13 | 오류 처리 | Git 미설치, 저장소 없음, 명령 실패를 사용자 행동 가능한 문장으로 안내한다. |
| F-14 | 저자 필터 | 패널 타이틀 아이콘 클릭 시 현재 파일 이력의 저자 목록을 다중 선택해 커밋 이력을 필터링한다. |

## Post-MVP 기능

| ID | 기능 | 설명 |
|----|------|------|
| F-15 | 최근 검색 저장 | 마지막 날짜 필터 조건을 `globalState`에 저장한다. |
| F-16 | 커밋 파일 트리 배지 | 변경 타입(추가/수정/삭제)을 파일 항목에 표시한다. |
| F-17 | 커밋 메시지 검색 | 커밋 이력 패널에서 키워드로 커밋을 필터링한다. |

## 예외 처리

| 상황 | 사용자 메시지 |
|------|---------------|
| Git이 설치되지 않음 | Git이 설치되어 있지 않습니다. |
| Git 저장소 없음 | Git 저장소를 찾을 수 없습니다. |
| 커밋 이력 없음 | 이 파일의 커밋 이력이 없습니다. |
| 변경 파일 없음 | 이 커밋에서 변경된 파일이 없습니다. |
| Git 명령 실패 | 오류 내용과 함께 재시도 버튼을 표시한다. |
| 이전/이후 커밋 없음 | diff 버튼을 비활성화한다. |
